# Sandeep Vijayarao Portfolio

This repository contains the source code for my personal portfolio website.

## Features
- **React + Vite**: Fast and modern web development stack.
- **Responsive Design**: Built to look great on desktop devices.
- **GitHub Actions Deployment**: Automatically deploys the codebase to GitHub Pages.

## Usage
To clone and run locally:

```bash
git clone https://github.com/sandeepvijayarao09/SandeepVijayarao.git
cd SandeepVijayarao
npm install
npm run dev
```
